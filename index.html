Clarifina 2024 ShellHacks Vanguard Winner

React, Node, Express, AWS (S3 and Lambda), TypeScript, Firebase, OpenAI API

Clarifina is a financial literacy web application designed with underrepresented communities in mind. Its core functionality enables users to upload financial documents, which are scanned to identify complex financial terminology. These terms are highlighted, allowing users to click on them for clear explanations. Additionally, a document summary with opportunities are provided.

Key Features

  • Leveraged AWS S3 to store PDFs and utilized AWS Lambda with Textract for coordinates to generate highlighted bounding boxes around key terms within documents.
  • Integrated the OpenAI API within an Express backend to analyze extracted text, both assesing complexity and providing explanations
  • Included user profiles, authentication, and past document viewing through Firebase

Gainesville Crime

Flask, Python, JavaScript, HTML/CSS, Leaflet API

A web application that visualizes crime data in Gainesville using an interactive heatmap. Users can filter the data based on various criteria to explore specific patterns. The application also features a comparison of Heap Sort and Shell Sort algorithms, showcasing their performance through real-time calculation and visualization of sorting times.

Key Features

  • Operates on over 210k crime data points
  • Uses a mixture of Leaftlet API and My Maps Geocode API to create map UI and relate address to lattitude/longitude
  • Imported JSON data to allow for toggle display of SNAP, AED, and Blue Phone locations
